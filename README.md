Um exemplo de uma aplicação em linha de comando em Dart.

- Ponto de entrada em `bin/`.
- Código de biblioteca em `lib/`.
- Testes unitários em `test/`

Para executar a aplicação, basta clonar o repositório e dentro dele, no terminal, rodar `dart run`.
Para rodar os testes, dentro do repositório, no terminal, rodar `dart test`.
